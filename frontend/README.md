# FYP Archive — Web App (React + Vite)

## Run locally
```bash
npm install
cp .env.example .env   # default API URL is http://localhost:5000/api
npm run dev            # http://localhost:5173
```

## Build for production
```bash
npm run build          # outputs to dist/
npm run preview        # preview the production build locally
```

Make sure the backend API is running first. See the top-level README.md for details.
